Layouts de Cabeçalho e Navegação

<header>: CSS Grid

Este cabeçalho é um exemplo de um layout CSS Grid. Ele usa a propriedade display: grid; para criar uma estrutura de grade.

O cabeçalho tem três elementos filhos: <img>, <div> e <nav>

Esses elementos são itens de grade, uma vez que seu pai é um contêiner de grade e, portanto, esses elementos participam do layout de grade.

O cabeçalho usa as seguintes declarações CSS para criar o layout de grade: 

display: grid;
 grid-template-columns: 120px 1fr 1fr;
 align-items: center;

Como resultado, o cabeçalho é dividido em três áreas: uma área de logotipo, uma área de título e uma área de navegação que estão lado a lado.

  • display faz do cabeçalho uma grade
  • grid-template-columns define que existem três colunas, a primeira com um tamanho fixo de 120px e as demais compartilhando igualmente o restante do espaço
  • align-items alinha os itens da grade ao meio do cabeçalho (a grade)

<nav>: CSS Flexbox

Esta navegação é um exemplo de um layout CSS Flexbox. Ele usa a propriedade display: flex; para criar um layout flexível.

A navegação tem um elemento filho: <ul>, que deve ser o contêiner Flex porque seus elementos filhos (os itens da lista) precisam trabalhar juntos como itens flexíveis.

O ul com a classe de navegacao usa as seguintes declarações CSS para criar o layout flex: 

display: flex;
 list-style: none;

Como resultado, os itens de navegação são distribuídos uniformemente dentro da área de navegação.

  • display faz da nav um contêiner flexível
  • list-style remove a estilização padrão da lista

A propriedade flex do li (itens da lista) é usada nos itens da lista para controlar seu tamanho e como eles crescem ou encolhem dentro da nav.

flex: 1 1 auto;
